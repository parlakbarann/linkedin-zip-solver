## ✨ Features

-   **One-Click Solution**: Solves the game with a single click, no complex steps required.
-   **Smart Redirection**: Automatically redirects you to the game page if you're not already there.
-   **Secure and Effective**: Reads the solution directly from page data, ensuring it's always up-to-date and accurate.
-   **Realistic Simulation**: Interacts with the game using human-like mouse clicks.
-   **Instant Notifications**: Stylish and clear notifications keep you informed about the process.
-   **Modern Technology**: Built according to the latest Chrome Manifest V3 standards.

## 🚀 Installation

Follow these steps to install the extension:

1.  **Download the Repository**:
    *   Download the latest version of this repository from the [releases](https://github.com/parlakbarann/linkedin-zip-solver/releases) page and extract the `.zip` file into a folder.
    *   Alternatively, clone the repository using `git clone https://github.com/parlakbarann/linkedin-zip-solver.git`.

2.  **Load into Chrome**:
    *   Open Chrome and navigate to `chrome://extensions`.
    *   Enable **Developer mode** in the top right corner.
    *   Click on **Load unpacked** and select the folder where you extracted the extension files.
    *   The extension should now appear in your toolbar!

---

## 📖 How to Use

1.  **Go to LinkedIn**: Log in to your LinkedIn account.
2.  **Click the Extension Icon**: Click on the **LinkedIn Zip Solver** icon in your toolbar.
    *   If you are not already on the game page, the extension will automatically redirect you to `https://www.linkedin.com/games/zip/`.
3.  **Sit Back and Relax**: The extension will automatically solve the game within seconds. You will see a notification on the screen showing the progress of the operation.
